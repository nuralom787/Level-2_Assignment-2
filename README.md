## 1. Project Name & Live URL.

* Project Name: Vehicle Rental System
* Live URL: https://vehicle-rental-system-six.vercel.app

## 2. Features & Technology Stack.

* Node.js + TypeScript
* Express.js (web framework)
* PostgreSQL (database)
* bcrypt (password hashing)
* jsonwebtoken (JWT authentication)
* momentjs
* node-cron

## 3. Setup & Usage Instructions.

### You can simply clone the repository or download the zip file and then install the node_modules. Additionally, add the environment variables, and you're now all set.
